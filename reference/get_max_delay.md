# Get maximum delay from a reporting triangle

Get maximum delay from a reporting triangle

## Usage

``` r
get_max_delay(x)
```

## Arguments

- x:

  A
  [reporting_triangle](https://baselinenowcast.epinowcast.org/reference/reporting_triangle-class.md)
  object.

## Value

Integer indicating the maximum delay.
